When giving cyber security advice, or delivering cyber security solutions, the key is to tailor.

The same security controls will not work for all customers.
The same breach advice will not work for all breaches.
The same training will not work for all people.

If you are just repeating the same template each time, you're being lazy, to the point of professional mis-information.

I hate the advice Optus, ACSC, and Federal Police are putting out in regards to the Optus breach.

Hardening your own online presence would do jack shit. The one thing that wasn't stolen, passwords, is the one thing that would be easiest to replace.

This is entirely on Optus, and no matter how "sophisticated" (must have been the same actors that targeted the ANU) the actors were to change their IP address (ooooh scary)...

YOU LEFT AN UNSECURED API ENDPOINT ON THE INTERNET WITH THE ABILITY TO PREDICT KEYS.

No amount of password managing or staying wary to emails from strangers is going to help the layman from that.
